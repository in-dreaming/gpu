# 分层

我建议分成五层。

## Layer0 Platform

SDL3

只负责：

```cpp
Window
Input
Clipboard
DragDrop
IME
Gamepad
Audio
```

不要负责：

```cpp
Graphics
Shader
Render
```

---

## Layer1 Shader

核心：

Slang

统一使用：

```slang
.slang
```

作为唯一 Shader Source。

禁止：

```text
glsl
hlsl
metal
wgsl
```

直接进入项目。

---

输出：

```text
SPIRV
DXIL
MSL
WGSL
```

---

同时负责：

```text
Reflection
TypeInfo
ParameterBlock
```

# Layer3 Framework

真正重要的是这层。

大部分引擎价值都在这里。

---

## Material System

基于 Slang Reflection 自动生成。

例如：

```slang
struct PBRMaterial
{
    Texture2D BaseColor;
    float Roughness;
}
```

```cpp
MaterialSchema
Inspector
Serialization
TS Binding
```
